## Getting Started

### Prerequisites

- Node.js (v16 or higher)
- React Native CLI
- Xcode (for iOS development)
- CocoaPods

### Installation

1. Install dependencies:
```bash
npm install
```

2. Install iOS dependencies:
```bash
cd ios && pod install && cd ..
```

3. Run the app:
```bash
# For iOS
npm run ios

# For Android
npm run android
```
